¿CÓMO ACTIVAR JavaScript EN DIFERENTES NAVEGADORES?

Chrome
Para activar JavaScript en el navegador Chrome, proceda como se indica a continuación:

1. Haga clic en el icono del menú situado en la barra de herramientas del navegador.

2. Seleccione Configuración.

3. Haga clic en el enlace Mostrar opciones avanzadas....

4. En la sección "Privacidad", haga clic en Configuración de contenido.

5. Seleccione Permitir que todos los sitios ejecuten JavaScript en la sección "JavaScript".

Mozilla FireFox
Para activar JavaScript en el navegador Firefox, proceda como se indica a continuación:

1. Haga clic en el menú Herramientas en el punto Opciones.

2. En la sección Contenido activar la casilla de verificación "Activar JavaScript".

3. Guarde los ajustes pulsando cada vez la tecla de Aceptar.

Internet Explorer 8
Para activar JavaScript en el navegador Internet Explorer 8, proceda como se indica a continuación:

1. Selecciona Herramientas y luego Opciones de Internet.

2. Haz clic en la pestaña Seguridad.

3. Haz clic en el botón Nivel personalizado y desplázate a la sección Automatización.

4. Marcar Active scripting y Scripting de applets de Java. Haz clic en Aceptar.

NOTA: Si aparece un cuadro de confirmación, selecciona SÍ.

5. Haz clic en Aceptar Cierra la ventana y vuelve a cargar la página.

Internet Explorer 7
Para activar JavaScript en el navegador Internet Explorer 7, proceda como se indica a continuación:

1. Selecciona Herramientas y luego Opciones de Internet.

2. Haz clic en la pestaña Seguridad.

3. Haz clic en el botón Nivel personalizado y desplázate a la sección Automatización.

4. Marcar Active scripting y Scripting de applets de Java.

5. Aceptar todo y reiniciar Internet Explorer.
